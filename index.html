NewsMash



What is NewsMash?

NewsMash is an application that helps users get to the news they want quickly. NewsMash gets news from all across the web by using the services of NewsBot, using content from the New York Times, The Guardian, Twitter and Reddit. This keeps you in the loop with the latest stories, as they are breaking.


How does it work?

NewsBot (open-sourced) is constantly sniffing for news stories. NewsBot finds anything interesting, and then sees if it is newsworthy. Once the bot determines that a trend is good, it automatically classifies the trend, and then asks all its sources if there are any stories for that trend, and proceeds to neatly display those trends. This is a project that is made possible thanks to a lot of existing services. NewsMash is delivered over a Content Delivery Network to ensure that the load times are minimal. Fork NewsMash on Github if you want to help the news bot become better!
